## 🟣 EventFlow — flagship product

The events industry runs on WhatsApp screenshots and scattered spreadsheets. Venues take reservations in DMs, ticketing lives on five different platforms, and customer communication breaks constantly. Nobody has fixed this properly — so I'm fixing it.

**EventFlow** is a two-sided, **AI-powered platform** connecting event vendors (wedding halls, photographers, organizers) with customers planning weddings, engagements, and celebrations. Vendors get appointment management, analytics, and event calendars; customers get instant booking, live availability, and chatbot-assisted planning. Revenue comes from vendor packages and promoted listings.

Competitors solve these problems separately. EventFlow is end-to-end.

**Stack:** `Django` · `Next.js` · `PostgreSQL` · `Celery` · `Docker`
